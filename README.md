# 🚀 Estrutura de Dados com Python

Este repositório é o meu laboratório de estudos sobre algoritmos e eficiência de código. Aqui aplico conceitos de estrutura de dados e analiso a performance usando a notação **Big O**.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3
* **Ambiente:** Google Colab
* **Versionamento:** Git & GitHub

## 📂 Conteúdo do Repositório

### 1. Ordenação Básica (Bubble Sort)
- **O que aprendi:** Como funciona o algoritmo de troca mútua para ordenar listas.
- **Complexidade:** $O(n^2)$ - Ideal para entender o custo de laços de repetição aninhados.
- [Acessar código](./01_Bubble_Sort.ipynb)

### 2. Estrutura de Pilha (Stack)
- **O que aprendi:** Implementação do conceito **LIFO** (Last In, First Out). Simulei um histórico de navegação onde a última página visitada é a primeira a ser removida ao "voltar".
- [Acessar código](./02_Pilha_Historico.ipynb)

### 3. Busca Binária vs Sequencial
- **O que aprendi:** A diferença prática entre um algoritmo linear $O(n)$ e um logarítmico $O(\log n)$. No teste realizado, a busca binária encontrou o item em uma lista de 1 milhão de elementos quase instantaneamente.
- [Acessar código](./03_Busca_Binaria_vs_Sequencial.ipynb)

### 4. Listas Ligadas (Linked Lists)
- **O que aprendi:** Como criar uma estrutura de dados dinâmica onde cada elemento (Nó) aponta para o próximo. Diferente dos arrays, as listas ligadas não precisam de espaço contíguo na memória.
- [Acessar código](./04_Lista_Ligada_Simples.ipynb)
